# Hollow-Knight-Cheat

A utility for changing save files for the game Hollow Knight, based on decryption and encryption of files, written in Python 3.
It is a GUI (graphical user interface) for conveniently changing values.

Inspired by:
- [Hollow-Knight-Save-Crypto](https://github.com/nakami/Hollow-Knight-Save-Crypto.git) by nakami

## Requirements

If you want to run the Python script via Python you need to install the following package:
```
pip install wheel
pip install pycryptodome
pip install pillow
```
